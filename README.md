## Project info
```
Esse projéto é para mostrar dados de produtividade mais visuais com graficos e dados para apresentção e entendimento mais claro
fora que fica conectado com uma base de dados online para gravar novos dados e mostrar um consolidade mais amplo com filtros de data
```

## 🚀 Tecnologias Usadas nesse Projeto
- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
